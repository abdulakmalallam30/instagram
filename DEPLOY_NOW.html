🚀 Deploy Your Instagram Clone Online

Choose one of these free hosting options

✅ Your code is ready for deployment! All files have been committed to Git.

⭐ Option 1: Render.com (Recommended - Easiest)

Why? Free, simple, no credit card required

1 Create GitHub Account: Visit github.com/signup
2 Create New Repository: Go to github.com/new
  • Repository name: instagram-clone
  • Make it Public
  • Click "Create repository"
3 Push Your Code: Copy your repository URL from GitHub, then run in PowerShell:
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/instagram-clone.git
git push -u origin main
Replace YOUR_USERNAME with your actual GitHub username
4 Deploy to Render:
  • Go to render.com
  • Sign up with your GitHub account
  • Click "New +" → "Web Service"
  • Connect your instagram-clone repository
  • Keep all default settings
  • Click "Create Web Service"
5 Wait 2-3 minutes for deployment to complete
6 Your site is live! 🎉
  • Login: https://your-app.onrender.com/src/html/loginPage.html
  • View Credentials: https://your-app.onrender.com/src/html/viewCredentials.html
🚀 Go to Render.com Create GitHub Repo

Option 2: Railway.app

Why? Fast deployment, $5 free credit monthly

1 Push your code to GitHub (same steps as Option 1, steps 1-3)
2 Deploy:
  • Go to railway.app
  • Sign up with GitHub
  • Click "New Project" → "Deploy from GitHub repo"
  • Select your repository
  • Wait for deployment
  • Click "Settings" → "Networking" → "Generate Domain"
🚂 Go to Railway.app

Option 3: Fly.io

Why? More control, command-line deployment

1 Install Fly CLI: Run in PowerShell:
iwr https://fly.io/install.ps1 -useb | iex
2 Deploy:
fly auth login
fly launch
fly deploy
✈️ Go to Fly.io
⚠️ Important Notes:
  • Free tier apps may sleep after 15 minutes of inactivity
  • First load might take 20-30 seconds (cold start)
  • The credentials.json file will reset when server restarts on free tier
  • For permanent storage, you'd need a database upgrade
